Zen Space
## Concept
Zen Space is een interactieve rustruimte voor focus, slaap en mentale gezondheid.
De gebruiker kan rustgevende geluiden mixen, visuele animaties bekijken, ademhalingsoefeningen doen en reflecteren via een notitiescherm.
Het idee is om een digitale plek te maken waar je even kan pauzeren van je dag. Je kiest zelf welke geluiden je wil horen, welke visual je wil zien, en hoe lang je wil rusten via de timer.
Pagina's

index.html — de hoofdpagina met alle interactieve features
about.html — uitleg over het project en de features
contact.html — contactgegevens van een partnerstudio via Fetch API

## Features

5 rustgevende geluiden (regen, zee, vogels, vuur, wind) met individuele volume sliders
Master volume slider
4 visuele animaties: Lava Lamp, Aurora Wave, Galaxy, Space Particles
Ademhalingsoefening met animerende cirkel
Rust timer (30 sec, 1 min, 2 min)
Random sfeer knop
Reflectie tekstveld met stemmingsdetectie
Willekeurige hero tekst bij elke paginalading
Contactpagina via Fetch API
Responsive design voor mobiel en desktop

## Technologieën

HTML, CSS, JavaScript
Three.js (Galaxy & Space Particles animaties)
Canvas API (Lava Lamp & Aurora Wave animaties)
Fetch API (contactpagina)
